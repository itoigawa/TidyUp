# TidyUp

https://tidyup-app.herokuapp.com/

## サービス概要
##### アイテムを最小限に
「服の数を減らしたい。」「無駄な出費を減らしたい。」ミニマルな暮らしを目指すアイテム管理サービスです

## サービスを作った思い
私は洗練されたシンプルなデザインやミニマルな暮らしを美しいと感じます。
しかし、似たような服を買ってしまう、着ないであろう服がクローゼットの中に眠っているという問題を抱えていました。
その問題を解決して服の数を減らしたい思い、手持ちのアイテムを管理して着る回数が少ないものを断捨離候補に上げるサービスを作りました。

 <!---普段の生活で思いの外、使わないものがあると感じていました。その中で服を減らしたいものの、似たような服を買ってしまう、着ないであろう服がクローゼットの中に眠っている。
そういった問題を解決したいと思い、手持ちのアイテムを管理して着る回数が少ないものを断捨離候補に上げるサービスを作りました。



コマンド rails _5.2.4.3_ new TidyUp --webpack=vue --database=mysql --skip-test --skip-turbolinks —skip bundle
 
## 登場人物
- エンドユーザー
  - 服の数を減らしたい方
  - 無駄な出費を減らしたい方
 
## ユーザーが抱える問題
クローゼットにある服を減らしたいものの、似たような服を買ってしまったり、着ないであろう服が眠っていたりしている。
 
## 解決方法
所有している服を管理する。
着る回数が少ないものを断捨離候補に。
 
## プロダクト
クローゼットの中身を可視化するWebアプリケーション
天気の情報取得
 
## マーケット
20-30代のミニマリストを目指す方。--->
